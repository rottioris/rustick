# rst-timer

Pomodoro timer application built with Tauri, React and TypeScript.

## Features

- Pomodoro timer with Focus, Short Break, and Long Break modes
- System tray support
- Native notifications
- Customizable settings
- Cross-platform (Linux, Windows, macOS)

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
